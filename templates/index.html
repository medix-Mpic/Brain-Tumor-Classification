Confusion Matrix

In the context of brain tumor classification, a confusion matrix is a table that compares the actual and predicted labels of the target variable, which could be glioma, meningioma, no tumor, or pituitary. The table displays the number of true positives, true negatives, false positives, and false negatives for each class, indicating how well the model has classified each type of tumor. For example, a high number of true positives and a low number of false positives for a particular class suggest that the model is good at identifying that type of tumor.

Model Performance

These are the evaluation metrics of a model's performance in classifying brain tumors into four classes: glioma, meningioma, no_tumor, and pituitary. The precision score measures the proportion of correctly classified instances among the predicted instances of each class. The recall score measures the proportion of correctly classified instances among the actual instances of each class. The f1-score is the harmonic mean of precision and recall, and it represents the model's accuracy for each class. The support column shows the number of instances of each class in the test set. The accuracy score measures the overall proportion of correctly classified instances. The macro avg and weighted avg show the average precision, recall, and f1-score for all classes.
